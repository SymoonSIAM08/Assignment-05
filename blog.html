Frequently Asked Questions

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of nodes, allowing you to manipulate the content and structure using JavaScript.

2. How do you select an element from the DOM?

You can select elements from the DOM using methods like `document.getElementById()`, `document.querySelector()`, and `document.querySelectorAll()`.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique that allows you to handle events at a higher level in the DOM rather than adding individual event listeners to each element. This is efficient, especially for dynamically created elements.

4. How do you manipulate an element's attributes and styles using the DOM?
